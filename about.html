⚡급할때.tools
글자수만나이·디데이연봉 실수령액

소개

급할때.tools는 회사원·대학생이 급할 때 자주 검색하는 작은 도구들을 한곳에 모은 무료 서비스입니다.

복잡한 설치나 가입 없이, 브라우저에서 바로 쓰도록 만들었어요. 모든 계산은 사용자의 기기 안에서만 처리되어 빠르고 안전합니다. 홈 화면에 추가하면 앱처럼, 오프라인에서도 쓸 수 있습니다.

현재 제공 도구

  • 글자수 세기 — 자소서·과제용 공백 포함/제외·바이트 카운터
  • 만 나이 · 디데이 — 생년월일로 만 나이, 목표일까지 D-Day
  • 연봉 실수령액 — 4대보험·세금 뺀 월 실수령액
  • 평수 변환기 — 평 ↔ ㎡, 가로·세로로 면적 계산
  • BMI 계산기 — 키·몸무게로 비만도·적정 체중
  • 비밀번호 생성기 — 안전한 랜덤 비밀번호
  • 이미지 도구 — 사진 압축·리사이즈·형식 변환(JPG·PNG·WebP)
  • PDF 도구 — 여러 PDF 합치기·페이지 나누기
  • 단위 변환기 — 길이·무게·온도·넓이·부피 변환
  • 퍼센트 계산기 — 할인가·증감률·비율
  • 더치페이 계산기 — N빵·1인당 금액·팁 정산
  • 로또 번호 생성기 — 1~45 무작위 6개
  • QR 코드 생성기 — URL·와이파이·전화 QR, PNG 저장
  • 배경 제거(누끼) — 흰·단색 배경을 투명 PNG로
  • 타이머·스톱워치 — 카운트다운 알람·랩 기록
  • 대출 이자 계산기 — 월 상환액·총 이자
  • 특수문자 모음 — 클릭 한 번 복사
  • 이미지 PDF 변환 — 여러 사진을 하나의 PDF로

실용 가이드는 블로그에서 볼 수 있어요. 도구는 사용자 의견을 반영해 계속 추가됩니다. 제안·문의는 운영자 이메일로 보내주세요.
