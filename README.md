# quick-tools — 급할 때 쓰는 무료 도구 모음

회사원·대학생이 급할 때 검색하는 작은 실용 도구들을 모은 **정적 웹사이트**.
서버·API·비용이 전혀 없고, 모든 계산이 **브라우저 안에서만** 처리됨. (애드센스 수익형)

## 들어있는 도구
| 페이지 | 도구 | 설명 |
|---|---|---|
| `index.html` | 홈 | 도구 카드 모음 + 소개 |
| `char-counter.html` | 글자수 세기 | 공백 포함/제외·단어수·바이트수(자소서용) |
| `age-calculator.html` | 만 나이·디데이 | 생년월일→만 나이, 목표일→D-Day |
| `salary-calculator.html` | 연봉 실수령액 | 4대보험·세금 뺀 월 실수령(참고용) |
| `area-converter.html` | 평수 변환기 | 평↔㎡ 양방향 + 가로세로 면적 |
| `bmi-calculator.html` | BMI 계산기 | 키·몸무게→비만도·적정체중(한국 기준, 참고용) |
| `password-generator.html` | 비밀번호 생성기 | crypto 난수 안전 비밀번호 + 강도 표시 |
| `privacy.html` / `about.html` | 필수 페이지 | 애드센스 승인 요건 |

## 어떻게 켜요?
설치 필요 없음. **`index.html` 더블클릭** → 브라우저에서 바로 열림.
(로컬에서 한 번에 보려면: 이 폴더에서 `python3 -m http.server` 후 `http://localhost:8000`)

## 특징 (차별점)
- 각 도구 페이지에 **SEO 메타 + `Schema.org FAQPage` JSON-LD** 삽입 → 구글 상위 + AI(ChatGPT·Perplexity) 인용(GEO) 노림.
- 모바일 반응형. 공통 디자인 `style.css`, 공통 헬퍼 `common.js`.

### v2 — 디자인·UX 개편 (경쟁사 대비 강점)
- 🎨 **브랜드 정체성**: 로고+⚡마스코트, 정돈된 컬러/타이포, 친근한 마이크로카피.
- 🌙 **다크모드 토글**: 설정 기억(localStorage) + 시스템 설정 기본값.
- ⚡ **즉시 체험**: "예시 넣기" 버튼 + 결과 강조 애니메이션(카운트업/플래시).
- 📲 **PWA(설치형 앱)**: 홈 화면 추가 + 오프라인 동작 (`manifest.json`, `sw.js`). 재방문↑.
- 🔁 **재방문 CTA(3곳)**: 상단 설치 배너 / 결과 직후 복사·저장 / 하단 관련도구·즐겨찾기 + 홈 "최근 쓴 도구".
- 🔒 **신뢰 배지**: "브라우저에서만 처리·가입 없음·무료" 상단 고정.
- 신규 파일: `manifest.json`, `sw.js`, `favicon.svg`, `icon.svg`.
- ⚠️ PWA 설치/오프라인은 **HTTPS 배포 후** 실제 동작(로컬은 등록까지). 아이콘은 SVG; iOS 대비 PNG(192/512)는 배포 시 생성 권장.

## 수익(애드센스) 준비
- 광고 자리(`.ad-slot`)만 미리 배치해 둠. **승인 후** 실제 애드센스 코드로 교체.
- 승인 요건: 유용한 도구 여러 개 + 소개/개인정보처리방침 페이지 + 오리지널 텍스트 → 충족.
- ⚠️ 저작권/다운로더류 정책과 무관한 **깨끗한 도구**라 정책 안전.
